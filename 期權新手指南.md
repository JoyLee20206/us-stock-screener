# 📖 美股期權新手指南

> 寫給「完全沒接觸過美股期權」的人。讀完這篇你會知道：什麼是 Call/Put、怎麼看期權價格、Greeks 是什麼、新手最容易踩的雷在哪。

---

## 1️⃣ 什麼是期權？三句話講完

**期權 = 一張「在未來某天，以特定價格買/賣股票的權利證書」。**

- 買的是「權利」，不是義務（你可以放棄不用）
- 為了這個權利，你付一筆錢叫**權利金 (Premium)**
- 一張期權合約代表 **100 股**（美股慣例）

---

## 2️⃣ 兩種基本型態：Call 和 Put

### 🟢 Call（看漲期權）
> 「我有權利在 X 月 Y 日以 $150 買 NVDA」

- 適合：覺得股票會漲
- 你**最多虧權利金**，股票漲越多賺越多

### 🔴 Put（看跌期權）
> 「我有權利在 X 月 Y 日以 $150 賣 NVDA」

- 適合：覺得股票會跌
- 你**最多虧權利金**，股票跌越多賺越多

### 損益圖（買方）

```
Long Call                          Long Put
P&L                                P&L
 │      ╱                           │ ╲
 │     ╱                            │  ╲
 │    ╱                             │   ╲
 │───╳─── 0 ─── 股價             ───╳────  0 ─── 股價
 │   │                             │    │
 │   BE                            BE
 │
 損失上限 = 權利金                   損失上限 = 權利金
```

## 3️⃣ 期權合約四要素

舉例：**NVDA 06/20 $150 Call @ $5.30**

| 要素 | 說明 | 例子 |
|------|------|------|
| 標的 (Underlying) | 是哪家公司 | NVDA |
| 到期日 (Expiration) | 哪一天權利失效 | 06/20 |
| 行權價 (Strike) | 約定的買/賣價 | $150 |
| 權利金 (Premium) | 你要付的價格 | $5.30 |

**這口合約你要付：$5.30 × 100 = $530**

如果到期日 NVDA 漲到 $160：
- 你的權利讓你能用 $150 買，再用 $160 賣
- 賺 $10 × 100 = $1,000
- 扣掉成本 $530，**淨賺 $470（報酬率 +89%）**

如果到期 NVDA 還在 $145（低於 $150）：
- 你的權利沒價值
- **賠光 $530**

## 4️⃣ 三個重要術語：ITM / ATM / OTM

以現價 $150 為例：

| 類型 | Call | Put | 比喻 |
|------|------|-----|------|
| **ITM**（價內） | Strike < $150 | Strike > $150 | 已經有「內含價值」，比較貴但中獎率高 |
| **ATM**（價平） | Strike = $150 | Strike = $150 | 剛好在邊緣 |
| **OTM**（價外） | Strike > $150 | Strike < $150 | 「賭機率」，便宜但容易歸零 |

**新手地雷：**買又便宜又遠 OTM 的 Call，看起來「賭中翻 10 倍」很誘人，**統計上 90% 會歸零**。

---

## 5️⃣ 期權價格 = 內含價值 + 時間價值

**NVDA $150 Call，現價 $155：**

- 內含價值 = $155 - $150 = **$5**（行使馬上能賺到的）
- 時間價值 = 權利金 - 內含價值 = **看 IV 和距離到期日決定**

**時間價值會隨時間流逝而消失（Theta 衰減）**，這就是買方最大的敵人 — 即使股票不動，你的期權也每天在貶值。

---

## 6️⃣ Greeks 三大要角（必懂）

把期權想成一台車：

| Greek | 意義 | 比喻 | 你想要 |
|-------|------|------|--------|
| **Delta (Δ)** | 股價 $1 變動 → 期權變多少 | 油門靈敏度 | Call: 0.5+<br>Put: -0.5- |
| **Theta (Θ)** | 每過一天，期權跌多少 | 油箱漏油速度 | 越接近 0 越好（時間夠） |
| **Vega (V)** | IV 變 1% → 期權變多少 | 風阻係數 | 進場時別買 IV 太高的 |

**買方公式：** 你需要 Delta 帶來的漲幅 > Theta 損耗 + IV 崩塌風險

### Delta 怎麼選？

| Delta | 含義 | 適合 |
|-------|------|------|
| 0.20-0.30 | 機率約 20-30% | 賭奇蹟、新手別選 |
| 0.40-0.50 | ATM 附近，性價比好 | 一般買方 |
| **0.50-0.60** ⭐ | 中獎率 50%、損益對稱 | **新手推薦** |
| 0.70+ | 已 ITM，貴但穩 | 取代買股票 |

## 7️⃣ 隱含波動率 (IV)：期權的「貴/便宜」

**IV 越高 → 期權越貴**

新手最容易踩的坑：在**財報前**買 Call。
- 財報前 IV 大漲（市場預期會跳）→ 期權貴
- 財報後 IV 暴跌（不確定性消失）→ 即使股票漲，期權也可能虧
- 這叫 **IV Crush（IV 崩塌）**

**怎麼判斷現在 IV 算貴？看 IV Rank**：
- IV Rank > 70% = 現在 IV 在過去一年的前 30% 高位 → 不適合買
- IV Rank < 30% = 現在 IV 算便宜 → 適合買

### 📊 本系統的雙軌設計

歷史 IV 資料 yfinance **不提供回填**，所以本系統採雙軌：

| 指標 | 何時顯示 | 含義 |
|------|---------|------|
| **IV Rank（N 日歷史）** | IV 累積 ≥ 30 天 | 真實 IV 百分位（最準） |
| **RV Rank（估算）** | IV 累積 < 30 天時 | 用「實現波動率」算的近似百分位 |

**RV（Realized Volatility）= 過去 20 日股價報酬的標準差年化**。它跟 IV 不一樣（IV 是市場「預期」、RV 是「實際發生」的），但在 70-85% 時間內方向一致，可作為過渡期間的替代品。

**判讀方式相同：**
- ≥ 70% → 🔥 偏貴（買方等等）
- 30-70% → 中等
- < 30% → ✅ 便宜（買方時機）

## 8️⃣ 新手 7 大死亡陷阱（一定要避開）

| ❌ 錯誤 | 後果 | 正確做法 |
|--------|------|---------|
| 買 OTM Delta < 0.3 | 90% 歸零 | 選 Delta 0.5+ |
| 買到期 < 14 天 | Theta 每天吃 5-10% | 選 21-45 天 |
| 在 IV Rank > 70 買 | 行情走對也虧 | 等 IV 降 |
| 不算盈虧平衡 | 漲了還是賠 | 進場前一定算 BE |
| 跨財報日 | IV crush 風險 | 避開或選財報後 |
| 流動性差 (OI<100) | 想出場沒人接 | 看 OI、Bid/Ask 差 |
| All-in 一口 | 一次歸零本金沒了 | 單筆 ≤ 總資金 2% |

## 9️⃣ 美股期權實務細節

| 項目 | 說明 |
|------|------|
| **交易時間** | 美股盤中 09:30-16:00 ET（台灣 21:30-04:00） |
| **一口** | 100 股 |
| **行使方式** | 美式（隨時可行使，幾乎都用「平倉賣掉」而不是真的買 100 股） |
| **行權價間距** | 高價股 $5、低價股 $1、超便宜股 $0.50 |
| **常見到期** | 週選（每週五）、月選（每月第三個週五）、季選、LEAPS（年選） |
| **手續費** | $0.65/口（IB、Fidelity 等），加上交易所費 $0.10-0.20 |
| **行使費** | 若實際行使約 $5-15，所以**盡量不要持有到到期**，到期前平倉 |

## 🚨 風險警語

期權買方統計上：
- 約 **70-80% 的買方合約最終歸零**
- 主因：時間價值衰減 + IV crush
- 賺錢的買方通常專注於：**方向判斷準** + **時機選對** + **嚴格停損**

**本系統提供的工具是輔助判斷，不構成投資建議。請務必：**
1. 先用模擬帳戶練習至少 1-2 個月
2. 真實交易單筆風險 ≤ 總資金 2%
3. 進場前算清楚最大虧損你能不能接受
4. 設定停損並嚴格執行
